# GitHub Activity CLI

A simple command-line tool to fetch and display GitHub user activity.

## Description

This CLI tool allows you to fetch and view recent activity for any GitHub user directly from your terminal.

Project inspired by [roadmap.sh](https://roadmap.sh/projects/github-user-activity)

## Requirements

- PHP 7.4 or higher
- Composer

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/mohabzalat22/github-activity-cli.git
   cd github-activity-cli
   ```

## Usage

Run the CLI tool with a GitHub username:

```bash
./index.php github-activity <username>
```

### Example

```bash
./index.php github-activity mohabzalat22
```
